# ΔE 阈值标定记录（Story 4.5）

**关联**：PRD FR-9 / Open Q3「动工前必做」；架构 AD-8（单一常量来源）；SM-C1 反指标。
**单一来源**：`thresholds.ts` 的 `DELTA_E_THRESHOLDS`。

## v1 定稿值

| 分级 | 区间 | 语义 |
|---|---|---|
| low | ΔE < 2 | 肉眼几乎无差 |
| mid | 2 ≤ ΔE ≤ 5 | 仔细看可辨 |
| high | ΔE > 5 | 明显不同色 |

## 依据

采用 CIEDE2000 色差领域公认经验区间，未做收窄/放宽：

- ΔE ≈ 1：恰可察觉差异（JND）的量级。
- ΔE < 2：通常被视为「极接近、非专业观察难辨」。
- ΔE 2–5：可察觉但仍属同色系范畴，适合作为「可接受替代」区间。
- ΔE > 5：明显色差，作为「高视觉影响」告警线，喂给 SM-C1 反指标。

这些阈值对拼豆替代场景稳健：替代目标取自同一品牌色板内的最近色，ΔE 分布通常集中在低/中区间，high 作为「替代质量劣化」的护栏。

## 待办（非阻断）

- **真实图纸人工复核**：维护者（第一验收人）用几张真实初音/风景图，在库存模式下跑替代，主观核对「低/中/高」分级与肉眼感受是否吻合；如需调整，仅改 `DELTA_E_THRESHOLDS` 字面量。
- 复核后可据替代结果统计 SM-C1：高视觉影响（ΔE > high）替代占比，确认未为求「全绿」无上限放大平均 ΔE。

FR-7/8/9 已基于稳定 API 实现，调整阈值零回炉。
